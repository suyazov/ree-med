# Архив проекта

Каноническое состояние темы ТриМед хранится в этом репозитории.

## Production

- Сайт: https://tree-med.ru
- Тема WordPress: `trimed`
- Контент страниц хранится в базе WordPress и редактируется через ACF.
- Исходники темы, локальная регистрация ACF-полей и все используемые статические assets находятся в Git.

## Что не хранится в Git

- пароли, SSH-ключи, API-токены и `.env`;
- база данных WordPress;
- пользовательские загрузки из `wp-content/uploads`;
- кэш, логи, временные browser-профили;
- воспроизводимые скриншоты визуальных проверок.

Перед удалением локальных рабочих копий production-тема была сверена с репозиторием по контрольным суммам. Временные каталоги `trimed-*-check`, `trimed-*-audit`, preview и старые рабочие копии не являются источником истины.

## Восстановление темы

1. Клонировать `https://github.com/suyazov/ree-med`.
2. Разместить содержимое репозитория в `wp-content/themes/trimed`.
3. Активировать тему и Advanced Custom Fields.
4. Подключить существующую базу WordPress и каталог `wp-content/uploads` из отдельного резервного хранилища.
